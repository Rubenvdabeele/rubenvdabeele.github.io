He strongly enjoys tackling new code bases, frameworks and tools, to deliver quality work based on readable code. He is a teamplayer and a strong communicator, comfortable with Agile workflows. Projects
MAGNETRON: What started as a colaboration bringing together code and strong visual language, MAGNETRON is a project bringing live audiovisualization to events. Emergent visuals that compound the auditory experience. Using a custom interface with midi support and live audio input, MAGNETRON has livened up multiple events over the past 2 years, and more to come. The project is an outlet for technogical experiments, out of the box problem solving (such as our all important 30m hdmi cable) and an important source of learning. Currently the Javascript/Electron based codebase is being converted to a native C++ application for desktop.
Javascript, Three.js, Electron, NodeJS, C++, GLSL, BGFX, IMGUI, GLFW, FFMPEG, CMAKE, Premake
NEYBUR: NEYBUR is Ruben's musical outlet. A slowly simmering project where analogue samples and digital instruments are brought together in downtempo songs. - Javascript, Three.js, Ableton
www.neybur.live
